=== Live Script Support for MATLAB® WordPress Publishing Tool ===

Contributors: Cheng Chen
Requires at least: WordPress 4.7
Tested up to: 5.8


This plugin adds support to your WordPress blog when you are using MATLAB® WordPress Publishing Tool.


== Description ==

The plugin will:
1. Leverage MathJax to render equations and formulas visually.
2. Provide extra style to display live script blog post appropriately 
3. Allow your WordPress blog media library to accept .mlx file

== Installation ==

Nothing unusual here!
